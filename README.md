# 💰 Smart Expense Tracker (AI Powered)

A simple AI-powered expense tracking system built with Python and Streamlit.  
It allows users to add expenses, automatically categorize them using a machine learning model, and analyze spending patterns with visual charts and budget predictions.

---

## 🚀 Features

- ➕ Add expenses with description and amount
- 🤖 AI-based expense category prediction
- 📋 View and manage expense history
- ✏️ Edit and delete expenses
- 📊 Monthly analytics dashboard
- 📈 Category-wise spending charts
- 💰 Budget prediction for next month
- 🕒 Stores date & time for each transaction

---

## 🛠️ Tech Stack

- Python 3.x
- Streamlit
- Pandas
- Scikit-learn
- Matplotlib

---

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/your-username/smart-expense-tracker.git
cd smart-expense-tracker
```
---

### 2. Create virtual environment
```bash
python3 -m venv venv
```

---

### 3. Activate virtual environment
```bash
venv\Scripts\activate
```

---

### 4. Install dependencies
```bash
pip install -r requirements.txt
```

---

### 5. Run the application
```bash
streamlit run app.py
```
